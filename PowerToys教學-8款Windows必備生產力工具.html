微軟 PowerToys

重塑 Windows 體驗的免費神器：精選 8 款必備工具，釋放您的終極生產力。

🚀 文章摘要

PowerToys 是一套微軟官方提供的免費系統工具集，其歷史可追溯至 Windows 95 時代。現今，它已演變為 GitHub 上的開源專案，並透過 Microsoft Store 發布。它不再是零散的工具，而是整合成一個統一的應用程式，讓使用者能輕鬆啟用、停用和管理多達 27 種（且仍在增加）的實用功能，深度客製化 Windows 體驗。

精選 8 款必備工具

🚀

PowerToys Run

類 macOS Spotlight 的快速啟動器 (Alt+Space)。不只啟動應用，還能搜尋檔案、執行 Shell 命令、網頁搜尋甚至數學運算。

👁️

Peek

類 macOS Quick Look 的檔案快速預覽 (Ctrl+Space)。選中檔案，一鍵查看圖片、PDF、文件的內容，無需開啟。

✍️

Text Extractor

強大的螢幕 OCR 工具 (Win+Shift+T)。可立即擷取螢幕上任何可見文字（來自圖片、影片或應用程式）並複製到剪貼簿。

🖼️

Image Resizer

整合至檔案總管右鍵選單的圖片批次處理工具。可快速、大量地調整一張或多張圖片的尺寸，無需開啟編輯軟體。

🗣️

Quick Accent

簡易的特殊字元輸入法。當需要輸入重音符號時（如 "à"），只需長按 'a' 鍵再按空白鍵或方向鍵即可選取。

⌨️

Keyboard Manager

鍵盤重對應工具。可將單一按鍵（如 A->B）或快速鍵（如 Ctrl+C -> Win+V）重新定義為其他功能、按鍵或文字字串。

🖱️

Mouse Utilities

滑鼠輔助工具集。包含「尋找我的滑鼠」（連按 Ctrl 或搖晃滑鼠）和點擊醒目提示功能，非常適合大螢幕或簡報錄影時使用。

🖥️

FancyZones

進階的視窗布局管理器。允許使用者自訂螢幕分割區域，按住 Shift 拖曳視窗即可將其快速吸附到預設的「區域」中。

💡 結論

PowerToys 遠不止這 8 項工具，其他如 Hosts File Editor 等功能也極其實用。它持續在開發中（這也是其 "Preview" 標籤的由來），不斷加入新功能。對於任何追求效率的 Windows 使用者來說，這套免費工具集絕對是不可或缺的。
