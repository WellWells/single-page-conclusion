🎨 卡通渲染 (Toon Shading) 技術概覽

探索 3D 模型如何模擬 2D 手繪動畫風格 (三渲二) 背後的視覺與技術奧秘

📝 核心摘要

卡通渲染 (Toon Shading)，亦稱為賽璐珞渲染 (Cel-shading) 或三渲二，是一種非真實感繪製 (NPR) 技術。其核心目標是讓 3D 電腦生成圖像擺脫真實物理光影，呈現出如同 2D 手繪動畫或漫畫的視覺風格。這套技術主要透過兩大特徵實現：清晰的物體輪廓線與邊緣明確的色塊陰影 (即二分陰影)，共同構築出獨特的「三渲二」美學。

🤔 什麼是卡通渲染？

傳統 3D 渲染追求的是「真實感」，盡力模擬現實世界的光影、材質和物理規律。然而，卡通渲染反其道而行，它屬於非真實感繪製 (Non-Photorealistic Rendering, NPR) 的範疇。

其名稱「賽璐珞渲染 (Cel-shading)」源自傳統動畫製作流程。動畫師會將角色繪製在透明的賽璐珞膠片 (Cel) 上，再將其疊加於背景之上。這種作畫方式造就了角色與背景分離、光影簡潔的獨特風格。卡通渲染的目標，就是利用 3D 技術重現這種手繪感與平面感。

🖊️ 核心技術 (一)：清晰的輪廓線

現實物體沒有「邊線」，我們透過光影過渡來感知其體積。但卡通和漫畫大量使用描邊，以低成本快速勾勒物體輪廓，並將其與背景清晰區分。在 3D 中生成描邊有多種方法：

  • 背面擴張法 (Backface Expansion)： 這是最經典且穩健的方法。系統會渲染模型兩次：第一次正常渲染；第二次將模型稍微放大，剔除正面 (只渲染背面)，並將背面塗成黑色。這樣，放大的黑色背面就會在物體邊緣露出來，形成統一粗細的輪廓線。
  • 預處理邊緣偵測 (Edge Detection)： 這種技術透過演算法分析模型。當相鄰兩個面的法線 (Normal) 角度差異超過一定閾值時 (例如模型表面的轉折處)，或是在深度、顏色上有明顯差異時，系統會自動在該處繪製線條。
  • 進階描邊 (如 本村線)： 為了讓線條更具手繪感，《罪惡裝備》系列改良了描邊技術。這種線條能根據視角方向和鏡頭距離動態變化：例如，當視線與物體表面接近 90 度時 (即輪廓邊緣)，線條變粗；正對表面時線條變細或消失。線條顏色也不再是純黑，而是能與光照結合，在亮處減弱、暗處加強，甚至與陰影融合。

🎨 核心技術 (二)：賽璐珞光影 (二分陰影)

卡通渲染的第二大特徵是違反物理規律的硬邊陰影，也稱為「二分陰影」。

在真實渲染中，光照強弱是根據物體表面法線與光線方向的夾角計算出來的，光影之間有平滑的過渡。而在卡通渲染中，著色器 (Shader) 會設定一個簡單的光照閾值：

// 簡易邏輯：
計算光照角度 (L·N)
如果 (光照角度 > 0.5) {
  顏色 = 亮部顏色
} 否則 {
  顏色 = 暗部顏色
}

透過這種方式，平滑的光影漸層被簡化為界線分明的亮暗兩塊色塊，從而實現了賽璐珞動畫般的質感。

😊 進階挑戰：生動的臉部陰影

二次元角色的臉部陰影若完全依賴模型自動產出，常會因為模型微小的起伏 (如鼻子、臉頰) 產生混亂或不具美感的陰影，導致「崩壞」。為了還原 2D 手繪中具有美感的特定陰影形狀 (例如臉頰上的一道陰影)，開發者採用了更精細的控制手段：

  • 模型拓撲與法線修改： 美術師在 3D 模型臉部「畫」出想要的陰影形狀 (透過調整模型布線)，並手動修改該區域內所有法線的方向，使其保持一致。這樣，當光線照射時，這個區域就會被判定為一個完整的平面，產生統一的明暗，形成固定的陰影形狀。
  • 距離場面部陰影 (Signed Distance Field, SDF)： 為了讓陰影能隨光影動態變化且更自然，部分技術採用了此方案。美術師先繪製好幾個關鍵的陰影形狀 (關鍵幀)，再依靠演算法在不同光照角度下，動態補足中間的陰影過渡形狀，達到絲滑流暢的效果。

📈 風格演進：從二分到豐富

隨著硬體效能提升，卡通渲染風格也在演進。以《原神》、《絕區零》等遊戲為代表，其風格已非傳統的二分陰影，而是走出了一條新路：

  • 臉部 2D，身體 3D： 角色臉部依然保持經典的二次元畫法 (大眼睛、小鼻子、平面化陰影)，以維持角色魅力。但身體和服裝則不斷增加細節，加入豐富的寫實材質 (如金屬、絲綢、皮革)，並使其呈現符合物理的光影表現。
  • Ramp 漸層貼圖 (Ramp Texture)： 為了擺脫單調的「亮部/暗部」二分色，現今的卡通渲染大量使用 Ramp 貼圖。這是一張由美術繪製的顏色漸層圖 (例如：從藍紫色的暗部 -> 固有色 -> 偏暖的高光)。渲染時，系統會用光照強度去這張貼圖上「取色」。這使得美術師能精確控制陰影的顏色 (如在暗部加入藍色) 和過渡的柔和度，讓畫面色彩更生動、更具手繪感。

🌀 結論：技術與藝術的螺旋

從攝影技術促使繪畫走向印象派與抽象派，到 3D 技術反過來模擬 2D 手繪的「假」。藝術與技術總是在彼此纏繞中協同進化。卡通渲染的發展，正是站在 3D 技術的肩膀上，再現並試圖突破 2D 的表現力，而這種進化在未來將會越來越有意思。
