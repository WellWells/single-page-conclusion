AI 動畫製作全攻略 從零到一的完整流程

本文件詳細拆解如何運用多種 AI 工具，從故事構思到最終剪輯，完成一部完整的動畫作品。

一、故事發想與角色概念設計

動畫的起點是故事，而角色的視覺設計則能確立整部作品的藝術風格，並讓故事人物更為立體。

核心工具

ChatGPT Gemini Midjourney Nano Banana (Gemini 2.5 Flash Image)

製作方法

  1. 激盪故事： 使用 ChatGPT 或 Gemini 共同激盪出故事雛形。
  2. 生成角色描述： 讓 AI 根據角色性格量身打造外貌特徵的提示詞。
  3. Midjourney 初步生成： 使用 Style Reference 或 Character Reference 等功能確立藝術風格。
  4. 確保角色一致性： 結合 Midjourney 的「Character Reference (--cref)」與 Nano Banana 的微調能力。
  5. 提升圖片品質： 對最終設計圖執行「放大 (Upscale)」以獲得高解析度版本。

二、分鏡腳本設計與合成

分鏡圖是動畫的藍圖，它將成為後續 AI 生成每個動畫片段的「起始影格 (Starting Frame)」。

核心工具

ChatGPT Midjourney Nano Banana Figma Upscayl

製作方法

  1. 生成分鏡腳本： 請 ChatGPT 詳細標示場景、角色表情與動作。
  2. 背景與人物分開處理： 先用 Midjourney 生成高品質背景，再用 Nano Banana 合成人物。為維持一致性，應搭配 Midjourney 的 Character Reference (--cref) 功能，並理解多次生成仍可能需要人工篩選微調。
  3. Nano Banana 合成技巧： 注意上傳順序以控制輸出尺寸比例。
  4. 建立「視覺提示」： 在 Figma 中用箭頭、註解等方式建立角色與場景的關聯。
  5. 分鏡圖畫質提升： 使用 Upscayl 將所有分鏡圖提升至高解析度。

三、AI 動畫生成與嘴型同步

將靜態的分鏡圖轉換為動態的影片，並讓角色的口白與嘴型同步。

Text to Video完全靠文字描述來生成影片。AI 創作自由度最大。用戶對畫面的控制力最低。
Image to Video提供一張「起始影格」，並用文字引導動態效果。適合已有靜態美術圖的使用者。動態效果的可控性有限。

六、版權、合規與倫理注意事項

在使用 AI 工具進行創作時，了解並遵守相關的法律與倫理規範至關重要。這不僅能保護您的作品，也能避免侵犯他人權益。

聲音複製 (Voice Cloning) 風險

使用 ElevenLabs 等平台的聲音複製功能時，嚴禁上傳未經授權的人物聲音，特別是公眾人物。這可能涉及肖像權、隱私權等法律問題。請務必使用您擁有合法授權的聲音來源。

AI 音樂版權爭議

使用 Suno 等工具生成的音樂，其訓練數據的版權問題仍在法律討論階段。例如，Suno 等平台曾面臨來自音樂版權方的訴訟。若要將 AI 生成的音樂用於商業發行，請務必詳閱該平台的服務條款，並了解潛在的版權風險。

圖像生成的肖像權

生成的人物圖像若與真人過於相似，也可能引發肖像權疑慮。在商業專案中，建議避免生成具有可識別真人特徵的圖像，或確保設計具有足夠的原創性。
