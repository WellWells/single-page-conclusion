  • 在工具失效時自動嘗試 Workaround。
  • 對複雜的 Git Worktree 操作展現高度理解。

UI 生成與美感躍進

安全性與行為模式 (SnitchBench)

在針對模型合規性與拒絕回答傾向的測試（俗稱 SnitchBench）中，Opus 4.5 展現了比前代更平衡的行為模式。

  • 減少過度拒絕： 相比於 Opus 4.0 經常因觸發安全機制而超時或拒絕回答，4.5 版本能更準確地理解使用者意圖並執行任務。
  • 告密傾向降低： 測試顯示，當面臨潛在的合規衝突文件時，Opus 4.5 向「媒體」或「政府」主動通報（Snitch）的機率顯著低於 Claude 4 Opus 與 Grok 系列，這意味著它更傾向於作為忠實的助手執行指令。

總結：值得升級嗎？

Opus 4.5 雖然價格不菲，但它是目前市場上「最令人放心的程式開發夥伴」。它的強項不在於生成速度，而在於極低的錯誤率與對複雜指令的深刻理解。如果你厭倦了需要反覆除錯 AI 生成碼的過程，Opus 4.5 絕對是目前最佳的投資。
